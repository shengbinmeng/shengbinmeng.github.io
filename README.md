# Shengbin's Blog

## Develop

Ensure Ruby and some build essentials are already installed (on Ubuntu):

	sudo apt install ruby ruby-dev build-essential

Install [jekyll](https://jekyllrb.com/) and other needed Ruby gems:

	gem install jekyll jekyll-feed jekyll-paginate jekyll-sitemap

Generate and serve the site:

	jekyll serve -w

## Deploy

This blog site is deployed as GitHub Pages: <https://shengbinmeng.github.io>, with custom domain: <https://blog.shengbin.me/>.
